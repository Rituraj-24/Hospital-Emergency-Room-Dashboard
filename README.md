# 🏥 Hospital Emergency Room Dashboard

## 📌 Project Overview

This project presents an interactive Hospital Emergency Room Dashboard built in Microsoft Excel to analyze key operational and patient-related metrics. The dashboard helps healthcare stakeholders monitor emergency room performance, identify trends, and support data-driven decision-making.

## 🎯 Objectives

* Analyze patient volume and emergency room utilization.
* Monitor average patient wait times.
* Track admission and non-admission rates.
* Understand patient demographics.
* Evaluate department referral patterns.
* Measure service efficiency through timely patient care.

## 🛠️ Tools & Technologies

* Microsoft Excel
* Power Query
* Power Pivot
* DAX (Data Analysis Expressions)
* Pivot Tables
* Interactive Dashboard Visualizations

## 📊 Key Performance Indicators (KPIs)

* Total Number of Patients
* Average Wait Time
* Patient Admission Status
* Patient Age Distribution
* Gender Analysis
* Department Referrals
* Percentage of Patients Seen Within 30 Minutes

## 🔄 Project Workflow

1. Data Extraction and Transformation using Power Query.
2. Data Modeling using Power Pivot.
3. Creation of DAX Measures and Calculations.
4. Data Analysis using Pivot Tables.
5. Dashboard Development with Interactive Visualizations and Slicers.
